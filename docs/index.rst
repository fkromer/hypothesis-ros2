hypothesis-ros2
===============

`hypothesis-ros2` provides data generators for message data fields, parameters
and common message fields of the `Robot Operating System 2 (ROS2)`_. `hypothesis-ros2`
enables *Property Based Testing* and *Fuzzy Testing* of *ROS2* nodes.

.. _Robot Operating System 2 (ROS2): https://github.com/ros2

.. warning::
    This repository is unmaintained and has been archived. Refer to `ros2_fuzzer`_ instead.

.. _ros2_fuzzer: https://github.com/aliasrobotics/ros2_fuzzer
